- EN 헤더: "How it sustains itself" / KR: "지속되는 구조"

  - Key Resources: 12인승 밴, 호스트(버디) 네트워크, 예약 플랫폼. / Twelve seat vans, the host network, this booking platform.
  - Value Proposition: 끊긴 구간 없는 여행과 기다림 없는 로컬 경험. / A trip with no missing links and no queue at the local door.
  - Customer Relationships: 라인 호스트 동승, 스탬프 재방문 루프. / A host on every ride, a stamp loop that brings you back.

  - Cost Structure: 차량 운영비, 호스트 인건비, 선주문 보증금. / Vehicle operations, host wages, pre-order deposits.
